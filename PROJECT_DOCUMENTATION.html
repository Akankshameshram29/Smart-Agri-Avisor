Operational Mechanism

01

Precise Geo-Tagging

The user navigates the responsive Leaflet map to their village geographical center. Upon tapping the screen, the system fetches high-precision coordinates (Lat/Lng) inside the India boundary.

02

Neural Site Search

The backend initializes a Hybrid Search Orchestration. It triggers native Google Search Grounding to find actual, real-time prices in local Mandis and identifies regional soil composition patterns using Gemini's knowledge core.

03

Advanced Forecasting

Unlike static apps, Smart Agri Advisor performs chronological analysis. It creates a 9-month window: 4 months of history, today's pivot rate, and a 4-month 2026 forecast based on upcoming weather and market volatility.

Developer Analysis & Demo Guide

Core Innovation: Hybrid Search

As a developer, you should highlight that Smart Agri Advisor doesn't just "talk" to an AI. It uses a Verification Pipeline. We use the Serper API/Google Search Tool as a deterministic layer—it fetches raw text from Mandi websites. The LLM (Gemini) acts as the reasoner, parsing this messy web data into a clean JSON structure for our UI.

  • • Zero Hallucination: Rates verified against live Google data.
  • • Adaptive Model Switching: Uses Gemini 1.5 for stability and 3.0 for complex deep-dives.

The 9-Point Temporal Engine

During the demo, mention our Chronological Consistency code. We don't hardcode "December" or "2025". We calculate the 4-month historical and 4-month 2026 projection window dynamically via Python's datetime.

Data Serialization Protocol
  • - T-4 to T-1: Verified Historical Trends
  • - T: Live Mandi Pivot (Today)
  • - T+1 to T+4: 2026 Bullish/Bearish Forecasts

Technical Resiliency: "What if APIs fail?"

Explain the Failsafe Recovery Logic in agent_service.py. If the live search fails, the agent falls back to its high-density internal training data and switches to providing "General Agricultural Resilience" advice instead of specific pricing, ensuring the app never breaks for the user.

Technical Overview

Complete architecture documentation for the AI-powered agricultural intelligence platform.

System Overview

Smart Agri Advisor is a full-stack agricultural intelligence platform that combines:

  • React 19.2.3 Frontend with Vite 6.2 for lightning-fast development
  • FastAPI 0.127.0 Backend with SQLAlchemy 2.0.36 ORM for persistent storage
  • Gemini 2.5 Flash & Gemini 3 Flash for AI-powered recommendations (google-genai 1.56.0)
  • Serper API for live web grounding of Mandi prices
  • Recharts 3.6.0 for beautiful price charts and data visualization

API Endpoints

Method Endpoint Description
POST /api/auth/login Login or register user
POST /api/analysis/run Run full crop analysis
POST /api/crops/details Get detailed crop insights
GET /api/history Fetch saved reports
DELETE /api/history/{id} Delete a report
POST /api/chat/ask Ask AI chatbot
GET /api/chat/history Get chat history

Database Schema

Users

  • id INTEGER PK
  • phone VARCHAR(15) UNIQUE
  • name VARCHAR(100)
  • joined_at DATETIME

ChatMessages

  • id INTEGER PK
  • user_id INTEGER FK
  • role VARCHAR(20)
  • content TEXT
  • timestamp DATETIME

FarmerRecords

  • id VARCHAR(20) PK
  • user_id INTEGER FK
  • lat, lng FLOAT
  • district VARCHAR(100)
  • data TEXT (JSON)

Key Features

🔄 API Key Rotation

Three Gemini API keys with automatic failover. If one exhausts quota, the system switches instantly.

GEMINI_KEYS=key1,key2,key3

🌐 Live Web Grounding

Serper API searches for real Mandi prices before generating recommendations. No hallucination.

_search_web(f"{crop} Mandi price today")

📍 Reverse Geocoding

OpenStreetMap Nominatim converts GPS coordinates to district names. Free, no API key required.

nominatim.openstreetmap.org/reverse

🧠 Neural Memory

AI chatbot receives full user context: name, past 5 searches, current screen, active report.

get_full_user_context(phone)

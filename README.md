## 🚀 Quick Overview
**Smart Agri Advisor** is a production-grade AI decision-support system for Indian farmers. It combines real-time Mandi search grounding with robust API failovers and a localized multi-linguistic advisor.

### 🌟 Key Features
- **Neural Persistence**: Chat history survives tab switches and browser refreshes via SQLite.
- **Script-Locking Engine**: Automatically matches user script (Hinglish/Latin vs. Hindi/Devanagari).
- **Dual-Unit Intelligence**: Standardizes Mandi prices to ₹/kg for farmer planning.
- **API Resilience**: Automated failover across multiple Gemini API keys.

### 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS, Recharts.
- **Backend**: Python FastAPI, SQLAlchemy, Google Gemini 1.5/2.5 Pro.
- **Data**: Serper API (Live Grounding), SQLite (Persistent Memory).

## ⚡ Quick Start

### 1. Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
